# Homebrew Tap

Personal Homebrew tap for macOS applications (ARM64 only).

## Saytype

Push-to-talk speech-to-text for macOS.

### Install

```bash
brew tap raphaelmitas/tap
brew install --cask raphaelmitas/tap/saytype
```

### Update

```bash
brew upgrade --cask saytype
```

### Uninstall

```bash
brew uninstall --cask saytype
```
